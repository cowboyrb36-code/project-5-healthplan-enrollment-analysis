# Project 5 – Health Plan Enrollment Analysis

## Overview
Analyzes health plan enrollment trends across demographic segments,
identifying seasonality patterns, gender balance, and market
concentration shifts from 2016 onward.

## Tools Used
- SQL (BigQuery) – Data cleaning, feature engineering
- Tableau – Four-worksheet interactive dashboard
- Excel – Initial data exploration

## Engineered Features
- Gender Ratio – Male/female share per segment
- HHI Concentration – Market concentration index
- Rolling 4Q Average – Smoothed enrollment trend
- Seasonality Index – Quarter-over-quarter pattern detection
- YoY Growth – Year-over-year enrollment change
